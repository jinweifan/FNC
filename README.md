# FNCViewer

[English README](./README.en.md)

FNCViewer 是一款跨平台 NC 程序查看、编辑与 3D 路径仿真的桌面应用，支持 Windows、Linux、macOS。

## 特色功能（重点）

### 激光一体机模式（核心）

激光一体机模式是 FNCViewer 的核心差异化能力，重点解决双面加工程序在同一界面中的可视化与校验问题。

- 支持正面坐标系：`XYZ`
- 支持反面坐标系：`UVW`
- `UVW` 路径使用独立颜色显示（图例中单独标识）
- 支持与代码编辑器、进度条、3D 拾取的完整联动
- 保留用户模式偏好（重启后记忆）

## 核心能力

- 打开并解析 `.nc/.anc` 程序文件
- 文件列表、代码编辑器、3D 视图联动
- 3D 路径拾取定位到代码行
- 播放、暂停、单步、速度档位、进度条拖动
- 视角切换、平移/旋转/缩放、回中心（`F`）
- 背景网格开关（`G`）
- 中英文切换（运行时即时生效）
- 多主题（跟随系统/浅色/藏蓝/深色）
- 最近打开文件（最多 10 个，重启恢复）
