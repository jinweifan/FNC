# First NC Viewer

[中文 README](./README.md)

First NC Viewer is a cross-platform desktop application for NC viewing, editing, and 3D simulation.

## Quick Start

- We recommend trying the sample files in `demo_nc/` first.
- Open `.nc/.anc` files from `demo_nc` to validate linkage, simulation, grid, and picking.

## Featured Capability (Key)

### Laser Integrated Machine Mode (Core)

Laser Integrated Machine mode is the most distinctive capability in First NC Viewer. It is designed for dual-side machining programs and synchronized verification.

- Front-side coordinate domain: `XYZ`
- Back-side coordinate domain: `UVW`
- Dedicated UVW path color in 3D legend
- Full linkage with editor, timeline/progress, and 3D picking
- User mode preference persistence across restarts

## Other Capabilities

- Open `.nc/.anc` files
- Linked file list, code editor, and 3D viewport
- Play/pause/step/speed/progress dragging
- View presets, pan/rotate/zoom, refocus (`F`)
- Grid toggle (`G`)
- Runtime i18n (`zh-CN/en-US`)
- Multiple themes
- Recent files (up to 10)
