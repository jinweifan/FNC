# First NC Viewer 产品文档

## 1. 产品定位

First NC Viewer 是一款面向雕刻机/CNC 场景的跨平台 NC 程序可视化工具，核心目标是：

- 快速查看和校验 NC 程序
- 在编辑与仿真之间建立实时联动
- 通过统一界面提升加工前检查效率

## 2. 核心特色：激光一体机模式（重点）

激光一体机模式是 First NC Viewer 的核心差异化能力，面向双面加工校验场景，支持在同一界面内直观区分正面与反面路径，降低翻面加工出错风险。

- 普通加工坐标：`XYZ`
- 激光一体机反面坐标：`UVW`
- `UVW` 反面路径使用独立颜色渲染并在图例单独标识
- 代码编辑器、3D 视图、进度条三端实时联动
- 模式设置会持久化，重启后自动恢复用户习惯

### 2.1 模式能力对比

- 通用雕刻机模式：面向常规 NC 加工，仅使用 `XYZ`
- 激光一体机模式：支持 `XYZ + UVW` 双坐标体系，重点用于反面路径核对

### 2.2 UVW 与 Z 方向规则

- `U -> X`
- `V -> Y`
- `W -> Z`（方向约定：`W > 0` 对应 `Z` 负方向，`W < 0` 对应 `Z` 正方向）

## 3. 目标用户

- 工艺工程师
- 操机师傅
- CAD/CAM 编程人员
- 售后与培训支持人员

## 4. 核心价值

- 减少程序错误带来的试切成本
- 提升程序审查效率
- 降低跨软件切换成本
- 提供跨系统一致的使用体验

## 5. 功能范围（当前版本）

### 5.1 文件与项目

- 打开 `.nc/.anc` 文件
- 最近打开文件（最多 10 条）
- 同目录文件浏览与切换
- 保存/另存为

### 5.2 编辑器

- 行号、语法高亮
- G/M/T/轴参数着色
- 代码折叠（基于抬刀/下刀段）
- 选中路径与代码行联动

### 5.3 3D 视图与交互

- 路径渲染（切削/快速移动/选中段）
- 激光一体机模式下支持 `UVW` 反面路径差异化显示
- 鼠标拾取路径 + 代码定位
- 平移、旋转、缩放、回中心
- 视角切换（Top/Front/Left/Right/Bottom）
- 背景网格开关（默认关闭）

### 5.4 仿真

- 播放/暂停/单步
- 低/中/高速档
- 进度条拖动
- 当前运行代码显示

### 5.5 系统能力

- 主题切换（跟随系统/浅色/藏蓝/深色）
- 中英文切换
- 布局与交互偏好记忆

## 6. 非功能要求

- 跨平台一致性（Windows / Ubuntu / macOS）
- 大文件场景下可用（高行数 NC 程序）
- UI 响应式布局
- 安全关闭与未保存提醒

## 7. 版本策略

- v0.1.x：核心查看、编辑、仿真与联动
- v0.2.x：性能深化、更多机型配置兼容
- v1.x：稳定发布与生态扩展

## 8. 已知边界

- 机床网络直连暂不在当前版本范围内
- 安装包生成依赖各平台构建环境与外网下载工具链
