# FNCViewer 帮助文档

## 1. 快速开始

1. 启动应用。
2. 点击“打开 NC”选择文件。
3. 建议优先使用仓库 `demo_nc/` 下示例文件（`.nc/.anc`）进行功能验证。

默认首次启动布局：代码编辑区 + 3D 视图（编辑区:3D 约 1:4）。

## 2. 文件与编辑

- 支持打开：`.nc`、`.anc`
- 文件列表支持：搜索、排序（创建时间/文件名/大小）
- 保存：`保存` / `另存为`
- 最近文件：自动记录最近 10 个

## 3. 3D 视图交互

- 左键拖动：平移
- 右键拖动：旋转
- 滚轮：缩放
- `F`：回到俯视并自动完整适配图形
- 网格按钮 / `G`：显示或隐藏网格（首次启动默认开启）

## 4. 联动能力

以下三者实时联动：
- 代码编辑器当前行
- 3D 当前高亮路径
- 底部进度条位置

支持：
- 点击 3D 路径定位到代码行
- `ArrowUp` / `ArrowDown` 在路径间跳转
- `Esc` 在 3D 视图中取消当前选中高亮

## 5. 主题与语言

- 语言：中文 / English（即时生效）
- 主题：跟随系统 / 浅色 / 藏蓝 / 深色

## 6. 激光一体机（特色）

- 支持 `XYZ` 正面路径
- 支持 `UVW` 反面路径
- 反面路径使用独立颜色图例
- 与代码、进度条、3D 选择保持联动

## 7. 常用快捷键（默认）

- `Alt+1`：显示/隐藏文件列表
- `Alt+2`：显示/隐藏编辑器
- `Alt+3`：显示/隐藏 3D 视图
- `F`：回中心并完整适配
- `1`：平移模式
- `2`：旋转模式
- `+`：放大
- `-`：缩小
- `G`：网格开关
- `H`：移动路径显示开关
- `ArrowUp`：上一段
- `ArrowDown`：下一段

## 8. 常见问题

### 8.1 双击 `.nc/.anc` 打不开或内容不加载

- 确认系统已将 `.nc/.anc` 默认打开方式设置为 FNCViewer
- 确认打开的是打包版应用（不是浏览器开发地址）

### 8.2 Ubuntu 安装报依赖或网络错误

请优先查看：`docs/packaging.md` 的 Ubuntu 章节，包含 DNS/镜像、dkms、AppImage 兼容问题处理。
