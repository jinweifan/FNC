# FNCViewer 帮助文档

## 1. 安装与启动

### 1.1 启动方式

- 直接双击应用程序启动
- 双击 `.nc` 文件（前提：系统已设置默认打开应用为 FNCViewer）
- 在应用内点击“打开 NC”

### 1.2 首次启动默认布局

- 默认展开：代码编辑区 + 3D 视图区
- 默认比例：约 `1:4`
- 后续会根据你的使用习惯自动恢复布局

## 2. 界面说明

### 2.1 菜单栏

- 打开 NC
- 保存
- 另存为
- 快捷键映射
- 模式、语言、主题

### 2.2 工具栏

- 重置
- 播放/暂停
- 上一步/下一步
- 速度档位
- 回中心
- 视角预设
- 平移/旋转
- 放大/缩小
- 网格开关
- 移动路径显示开关

### 2.3 左侧窄边栏

- 文件列表显隐
- 代码编辑器显隐
- 3D 视图显隐

## 3. 激光一体机模式（重点）

### 3.1 模式说明

- 通用雕刻机模式：常规 `XYZ` 加工
- 激光一体机模式：支持 `XYZ + UVW` 双坐标域
- `UVW` 反面路径会在 3D 视图中以独立颜色显示

### 3.2 UVW 方向规则

- `U -> X`
- `V -> Y`
- `W -> Z`
- 方向约定：`W > 0` 为 `Z` 负方向，`W < 0` 为 `Z` 正方向

### 3.3 推荐操作流程

1. 在菜单栏右侧切换到“激光一体机”模式
2. 打开 NC 文件，确认图例中出现 `UVW` 反面路径
3. 通过进度条拖动或播放检查正反面路径联动
4. 点击路径核对代码行，重点检查翻面关键段

## 4. 文件操作

### 4.1 打开文件

1. 点击“打开 NC”
2. 选择 `.nc/.anc`
3. 自动解析并显示路径

### 4.2 文件列表交互

- 单击：选中
- 双击：打开
- 支持按文件名搜索
- 支持按创建时间/文件名/大小排序

### 4.3 最近文件

- 自动记录最近打开的 10 个文件
- 启动后默认选中最近一个
- 若无启动参数文件，自动尝试打开最近文件

## 5. 代码编辑器

### 5.1 高亮规则

- G/M/T 指令着色
- XYZ/UVW 等轴参数着色
- 注释斜体

### 5.2 联动行为

- 3D 选中路径后，编辑器自动定位对应行
- 编辑器当前行可驱动 3D 当前高亮段

## 6. 3D 视图

### 6.1 鼠标操作

- 左键拖动：平移
- 右键拖动：旋转
- 滚轮：缩放

### 6.2 路径选择

- 悬停线段可出现 hover 提示
- 点击线段后高亮当前段并显示方向箭头
- 上下键可跳转上一段/下一段

### 6.3 视图控制

- 视角预设：Top/Front/Left/Right/Bottom
- 回中心：快捷键 `F`
- 网格：默认关闭，开启后显示工业风方格背景参照

## 7. 仿真与进度

- 播放/暂停/单步
- 低速/标准/高速
- 进度条可拖动
- 当前行号、路径类型和运行代码实时显示
- 激光一体机模式下可同时核对 `UVW` 反面路径

### 8.1 主题

- 深色

### 8.2 语言

- 中文（`zh-CN`）

切换后立即生效，不中断当前会话。

## 9. 快捷键（默认）

- `Alt+1`：显示/隐藏文件列表
- `Alt+2`：显示/隐藏编辑器
- `Alt+3`：显示/隐藏 3D 视图
- `F`：回到中心
- `1`：平移模式
- `2`：旋转模式
- `+`：放大
- `-`：缩小
- `G`：显示/隐藏网格
- `H`：显示/隐藏移动路径
- `ArrowUp`：上一段
- `ArrowDown`：下一段

你可以在“快捷键映射”里修改。

## 10. 常见问题

### 10.1 双击 `.nc` 没有自动打开

- 请先在操作系统中设置“默认打开应用”为 FNCViewer
- 确认文件后缀为 `.nc`

### 10.2 点击右上角关闭无响应

- 若存在未保存修改，会弹出保存确认
- 选择“保存”或“放弃修改”后即可关闭

### 10.3 网格按钮开启后看不到网格

- 已改为工业风背景方格
- 如仍不明显，请检查是否处于极端缩放状态，可先按 `F`

### 10.4 打包安装包失败

- 多数是网络下载 NSIS/WiX 依赖超时
- 建议改用 GitHub Actions 进行云端打包
